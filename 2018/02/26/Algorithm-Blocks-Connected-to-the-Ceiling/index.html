Algorithm: Blocks Connected to the Ceiling

Posted on 2018-02-26

Given a 2d array, imagine it represents the blocks on the wall. 1 means there is a block and 0 means there isn’t. The first line of blocks connected to the ceiling. The block could “pull” the block down next to it, and it could also support the block left or right to it.When we remove one of the blocks, how many blocks left?

For example, here is the matrix, there are total 11 blocks.

remove matrix[1][3], no other blocks drop except this one, return 10;

remove matrix[2][1], matrix[2][2] and matrix[2][3] drop, return 8;

Solution:

This could be seen as a graph. The incoming edge of the node means it is “supported” or “pulled” by its neighbor node. There are three conditions:

The indegree of a node is ZERO, which means there is no other nodes supporting it. It should drop.

The indegree of a node is ONE, which means it could not support back to the node that supports itself. we should remove such outgoing edge.

The indegree of a node is more than ONE, nothing happens.

In the first two cases, we should also check its neighbor nodes.
